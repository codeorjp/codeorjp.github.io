生徒の作品を確認する

ひとり一人の課題を確認する方法

  • 生徒一覧から該当生徒の「確認する」をクリック

  • 該当生徒の一覧が表示される

それぞれの課題のチェックや欠席の確認などが行えます。

課題ごとにまとめて課題を確認する方法

  • 評価をする際などは課題ごとにまとめて表示できます。

  • 10人ごとに画面表示されます。
  • 未提出の生徒は＜未提出＞と表示されます。
  • それぞれの課題について、拡大/縮小、中央に寄せる操作が可能です。
