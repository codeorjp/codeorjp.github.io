（1）Googe アカウントを1つ用意する。

※個人で作成したアカウント（〜@gmail.com）でも登録可能です。

生徒のアカウント登録にGoogle Classroomの名簿を流用したい場合は、教育委員会等から発行されている先生のアカウントをご用意ください。（〜＠xxxxx.ed.jp）

SSO（シングルサインオン）機能とは（別サイトへ移動します）

現在、SSOに利用できるのはGoogleのアカウントのみです。

（2）プログル技術のトップメニューへ

先生→新規登録をクリック

（3）「ソーシャルアカウントで新規登録/ログイン」をクリックする

（4）Googleアカウントで新規登録/ログインをクリックする

（5）利用するアカウントをクリックする

※必要に応じてGoogleアカウントのパスワードや2段階認証を求められます。

（6）ログインできると先生用の管理画面が表示される

先生アカウントが作れたら、次回からトップメニューの「ログイン」から入れます。

次はクラスの登録を行います。

新規にGoogleアカウントを用いて先生アカウントを作成する場合は画面下のBackで1ページ戻ってください。

メールアドレスとパスワードを用いて作られた先生アカウントに対し、Googleアカウントを紐付けることにより、ログイン時のパスワード入力を省略したり、Google Classroomからデータを引用して授業クラスを作成することができます。

管理メニュー → アカウント設定

アカウントを選択して、連携完了。

Google Classroomからインポートしないでクラス登録する場合はこちら。

「GoogleClassroomのクラスで登録」をクリックする。

インポートに利用する先生のアカウントを選択し、Googleアカウントへのアクセスを許可する。

インポート可能なクラスが表示されているので、インポートしたいクラスを選択する。

※オーナー（主担任）になっているGoogle Classroom 以外は表示されません。

生徒がインポートされ、クラスが作成されました。

※あとは必要なクラスを同じようにインポートしてください。

アカウントの選択

Google Classroomからインポートしたクラスを用いる場合は、生徒に初期パスワード等を配布する必要はありません。

生徒のログイン操作は以下の通りです。

生徒の「ログイン」をクリックする

「Googleアカウントでログイン」をクリックする

アカウントの選択をする。

クラスと出席番号の選択

生徒は参加するクラスをクリックする。

確認のメッセージが表示される。

出席番号を入力する。

※出席番号を誤って入力してしまった場合は、先生の管理画面で訂正してください。

クラスと出席番号を確認する。

出席番号を間違えた場合

生徒が出席番号を間違えた（重複した）場合は、以下のようなエラーが表示されます。

エラーが表示された生徒か、先に登録を終えた生徒のどちらかが出席番号を間違えています。生徒のメイン画面を表示して、出席番号が正しいか確認しましょう。

出席番号を誤って登録している生徒を選択します。

正しい出席番号に訂正してください。

※間違えた生徒の出席番号を修正できれば、正しい出席番号で登録が行えます。

ニックネームとアイコンの決定

ニックネームとアイコンを選択する。

ログインできました。

転出する生徒を削除する場合

転出する生徒の「ゴミ箱」をクリックする

転入する生徒を追加する場合

まず先にGoogle Classroom側で追加の作業を行います。

追加する生徒をGoogle Classroomで招待する。

該当する生徒をGoogle Classroomに登録後、クラスのメニューから「生徒データを再インポートする」をクリックする。

インポートするクラスを選びます。

転入生がインポートされます。

これで転入生は他の生徒と同様にログインから操作を行えます。

クラスの削除

削除したいクラスのゴミ箱マークをクリック

※クラスを削除すると生徒の学習データは利用できなくなりますので、ご注意ください。

※クラスを削除するとデータ復旧もできませんので、慎重に作業をお願いします。

※年度をまたいでも自動でクラスの消去は行いませんが、不正利用を防ぐために年度末3/31を過ぎると生徒アカウントにはロックがかかります。学年をまたいで指導に当たられる場合はご注意ください。

年度末に不要になったクラスは削除するようにしてください。

みんなのコードではGoogleのアカウントに関するパスワードの再発行等は行えません。学校のICT担当者または教育委員会のICT担当者へ御相談ください。

SSO機能を使って生徒のアカウントを作成した場合は、生徒はパスワードを用いてログインしないため、パスワードを間違えてロックがかかることがありません。

※ただし、年度末に不正利用を防ぐためにシステムが生徒アカウントにロックをかけます。年度をまたいで指導する場合は、先生の管理画面からロックを外してください。

登録されている先生の名前などは「アカウント設定」から変更できます。

みんなのコードからの配信メールを受信しない場合の設定やSSOの解除、アカウントの抹消（先生＆生徒データの全消去）もこちらから行えます。

※退会してしまうと生徒の授業に関するデータなどが利用できなくなってしまいます。十分注意してください。

勤務先の異動などで、先生の利用するGoogleアカウントに変更があった場合は、手続きが必要になります。

その際、以下のようなパターンが考えられます。

1：新しい勤務校でもGoogle Workspaceを利用しており、Googleアカウント並びにClassroomが利用できる。

旧勤務校のGoogleアカウント解除 → メールアドレス＆パスワード運用 → 新勤務校のGoogleアカウント連携 の操作が必要

2：新しい勤務校ではGoogle Workspaceを利用しておらず、Googleアカウント並びにClassroomが利用できない。

A)旧勤務校のGoogleアカウント → メールアドレス＆パスワードの運用に変更 → 個人作成のGoogleアカウント

B)旧勤務校のGoogleアカウント → メールアドレス＆パスワードの運用に変更

新しい勤務校でGoogleアカウントが利用できない場合も、個人で作成したGoogleアカウントとの連携は可能です。

教師用アカウントに紐付けされたGoogleアカウントを外す作業を行います。

※先生アカウントを作る際に、メールアドレス＆パスワードを用いて先生アカウントを作成し、SSOの手続きを行った方はこの作業は不要ですので、Nextで次のページへ進んでください。

まず、先生のアカウント設定の画面を表示します。

管理メニュー → アカウント設定

「パスワードを変更する」をクリックしパスワードを設定する。

パスワードの再設定

パスワードを入力してください。

※続いて、メールアドレスの設定を行います。

パスワードの設定が終わったら、「メールアドレスを変更する」をクリックして設定します。

「パスワードを変更する」をクリックしパスワードを設定する。

新しいメールアドレスを入力して「登録」を押す。

新しいメールアドレス先に登録確認メールが送信されます。

確認メールが届いていることを確認して、文中のURLをクリックする。

以上で、メールアドレスとパスワードの紐付け作業が完了です。

この後、一度ログアウトして、メールアドレスとパスワードにて先生アカウントにログインできることを念のため確認してください。

連携解除のボタンが有効になるので、クリックしてください。

新しい勤務校でGoogleアカウントが発行されたら、前述の既存アカウントに紐付けの説明を参考に登録を行ってください。
